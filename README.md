AI-Based Recommendation System

An AI-Based Recommendation System developed using Java and Eclipse IDE that recommends movies to users based on collaborative filtering concepts. The application reads user ratings from a CSV dataset and generates personalized movie suggestions by analyzing highly rated movies from other users.

This project demonstrates the basic working of recommendation engines used in platforms like Netflix, Amazon, and Spotify. The system avoids recommending already watched movies and provides more relevant suggestions based on user preferences.

Features:
Reads movie ratings from a CSV file
Personalized movie recommendations
Collaborative filtering recommendation logic
Avoids duplicate or already watched recommendations
Uses Java Collections Framework
Simple console-based implementation
Exception handling and file processing

Technologies Used:

>Eclipse IDE
>CSV File Handling

>HashSet

How It Works:
1.The system reads movie ratings from the CSV dataset.
2.It stores user ratings using Java collections.
3.The user enters a username.
4.The program identifies movies already watched by the user.
5.It recommends highly rated movies from other users.
6.Personalized movie recommendations are displayed with scores.
